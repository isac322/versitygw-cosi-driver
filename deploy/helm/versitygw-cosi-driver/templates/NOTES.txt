== Prerequisites ==

COSI controller and CRDs must be installed before using this driver.
If not already installed, run:

  kubectl create -k 'https://github.com/kubernetes-sigs/container-object-storage-interface//?ref=v0.2.2'

VersityGW must be configured with:
  - IAM enabled: iam.enabled=true (or --iam-dir flag)
    Without this, the Admin API's create-user returns XAdminMethodNotSupported.
  - Admin API enabled: admin.enabled=true (or --admin-port flag)
    The driver uses the Admin API to manage users.
